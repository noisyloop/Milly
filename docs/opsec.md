# OpSec Principles

## General
- Separate professional and personal identities
- Never mix research accounts with personal accounts
- Use separate browsers for different threat profiles
- Deploy canary tokens across public-facing infrastructure

## Infrastructure
- Never hardcode credentials
- Secrets in environment variables only
- Separate API keys per project, never reuse
- .env always in .gitignore

## Research
- Responsible disclosure before public release
- Document and timestamp all findings
- Never test on systems without explicit permission

## Communications
- Assume anything on public platforms is public
- Sensitive research discussions off mainstream platforms
